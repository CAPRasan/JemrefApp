## なぜこのアプリケーションが必要なのか

1. **従来の文献管理ツールは英語圏の文献をベースにしている**   
　文系の和文文献になじまない。結局、Excelや手書きのメモ、または記憶頼りになる。  
2. **大学教員の多様化・多忙化が進み、業績を求める圧力が高まっている**  
　玉石混交の論文が累積。それを適切に評価・整理することは、現代研究者の大前提。  
3. **「大学人こそが真の研究者」というが、文系では大学の研究環境が年々劣化している**  
　大学外でも本格的に研究できる環境を整え、文系研究を存続・活性化したい。

 **文献を〈忘れない〉、〈見逃さない〉、〈積み残さない〉** ための充実したツールを、</br>
 プロ・アマ問わず、気軽に使えるようにすることが求められます。

## 利用方法

※現時点ではローカル環境でのみ動かすことができます。

1. アカウント作成、またはログインする。

2. 書誌情報を、その形態（著書、雑誌論文など）に応じて入力する。

3. 書誌情報を一覧画面で検索・更新する。

# 環境設定

バージョン情報

Ruby: 3.3.4

Rails: 7.2.1

## 1. Ruby on Railsの環境構築方法

### 注意

- ここでは、Mac OSでゼロから構築するケースを想定しています。
- インストール済みの方は、必要に応じて読み替え / 読み飛ばしたり、rbenv、Railsなどを更新したりしてください。

### (1) Homebrewのインストール  
* 次のリンクから、Homebrewをインストールするためのコマンドを入手します。  
https://brew.sh/ja/   
* 入手したコマンドを入力・実行してください。たとえば次のようなものです。  
(インストールに時間がかかることがあります。)
~~~
 /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
~~~


* 次のコマンドを入力し、Homebrewが正しくインストールされたか確認してください。
~~~
brew -v 
~~~
### (2) rbenvのインストール

* 次のコマンドを入力し、rbenvをインストールしてください。
~~~
brew install rbenv
~~~
* rbenvが正しくインストールされたことを確認するため、次のコマンドを実行します。
~~~
rbenv -v
~~~
* rbenvにパスを通すため、次の3つのコマンドをそれぞれ入力・実行します。   
設定ののち、ターミナルを再起動してください。
~~~
echo 'export RBENV_ROOT="$HOME/.rbenv"' >> ~/.zshrc
~~~
~~~
echo 'export PATH="$RBENV_ROOT/bin:$PATH"' >> ~/.zshrc
~~~
~~~
echo 'eval "$(rbenv init -)"' >>  ~/.zshrc
~~~

### (3) Rubyのインストール

* 次のコマンドを実行し、Rubyをインストールしてください。
~~~
rbenv install 3.3.4
~~~
* 次のコマンドを実行し、Rubyのバージョン設定をしてください。
~~~
rbenv global 3.3.4
~~~
* Rubyが正しくインストールされたことを確認するため、次のコマンドを実行します。
~~~
ruby -v
~~~
### (4)Railsのインストール

* 次のコマンドを実行し、Railsをインストールしてください。
~~~
gem install rails -v “7.2.1”
~~~
* Railsが正しくインストールされたことを確認するため、次のコマンドを実行します。
~~~
rails -v
~~~
## 2. JemRefのインストール方法

### (1) Jemrefリポジトリのクローン
下記コマンドを実行してください。
~~~
git clone https://github.com/CAPRasan/JemrefApp.git
~~~

### (2) Gemのインストール
JemRefのディレクトリに移動し、下記コマンドを実行してください。

~~~
bundle install
~~~

### (3) データベースの設定
データベースを作成・マイグレーションするため、下記コマンドを順に実行してください。
~~~
rails db:create
~~~
~~~
rails db:migrate
~~~

### (4) ローカルサーバーを起動
下記コマンドを実行してください。
~~~
rails s
~~~

### (5) Webブラウザを起動
下記URLにアクセスしてください。  
[http://localhost:3000](http://localhost:3000/)
